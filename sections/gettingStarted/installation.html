Installing Mooltipage

Prerequisites:

Mooltipage is based on Node.JS and requires version12.0or later to run. If you do not have Node.JS, then you can download it at no cost fromthe Node.JS official website.
No other software is required.

Installation:

Mooltipage is designed to be installed and run through NPM, a package management tool that is included with Node.JS. Follow these steps to install Mooltipage via NPM. If you are installing into an existing NPM project, then skip to step #4.

  1. Create a new folder somewhere on your PC. This is where your web project will be located.
  2. Open a Terminal, Command Prompt, or PowerShell window to the newly created folder.
  3. Execute the commandnpm init. Follow the prompts to complete project setup.
  4. Execute the commandnpm install --save-dev mooltipage.

Congratulations, Mooltipage is now installed and ready for use. For basic usage instructions, please proceed toGetting Started - CLIorGetting Started - JS API.

Getting Started with Mooltipage

This guide covers the basic installation and usage of Mooltipage for both the CLI and JavaScript API. To jump to a specific topic, use the links in the sidebar.

Mooltipage includes two primary modes of operation: the CLI - or command line interface - and the JavaScript API. Before you begin, you must decide which mode you intend to use. Generally, the CLI is faster and easier to set up but the API provides more low level control and advanced functionality.

Once you have decided which Mooltipage mode is right for you, then move on to theinstallationsection.
